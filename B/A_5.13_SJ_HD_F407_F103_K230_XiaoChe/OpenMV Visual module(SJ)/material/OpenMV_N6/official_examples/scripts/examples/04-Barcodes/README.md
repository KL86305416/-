# 04-Barcodes — 条码/二维码识别

| 例程 | 说明 |
|---|---|
| `find_barcodes.py` | 一维/二维条码检测与解码（EAN、UPC、Code128、QR 等） |
| `find_barcodes_adaptive.py` | 自适应阈值条码检测（应对不均匀光照） |
| `find_datamatrices.py` | Data Matrix 码检测 |
| `find_qrcodes.py` | QR 二维码检测与解码 |
| `read_barcodes.py` | 读取条码并输出详细信息（类型、数据、坐标） |

---

**N6 注意**：N6 可全分辨率运行条码检测，高分辨率下识别更准确。
