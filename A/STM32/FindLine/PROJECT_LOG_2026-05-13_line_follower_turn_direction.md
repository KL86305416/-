# 项目日志：仅修正 0x80 循迹转向方向

- 时间：2026-05-13 13:00
- 改动范围：循迹控制中 `0x80` 单点数据的误差方向

## 问题现象

- 原本只有灰度传感器数据为 `0x80` 时方向异常：输出左轮速度高、右轮速度低，例如左轮 `163`、右轮 `35`。
- 其它传感器数据的转向方向没有问题，不应整体反转左右轮差速公式。

## 改动内容

- 恢复原来的左右轮速度分配公式，避免影响其它传感器数据。
- 仅当 `gray_data.value == 0x80U` 时，将该次 `line_position` 转为负方向。
- 这样 `0x80` 会得到右轮更快、左轮更慢的差速，车辆按预期向左修正。

## 验证结果

- 已执行 `cmake --build build/Debug`，编译和链接通过。
- 构建结果：RAM 使用 `3232 B / 32 KB`，FLASH 使用 `21716 B / 128 KB`。
- 上车测试时重点观察 `0x80` 单点状态，确认右轮速度高于左轮速度。
