# 项目日志：蓝牙串口波特率提升到 115200

- 时间：2026-05-13 13:31
- 改动范围：蓝牙串口配置、OLED 蓝牙调试页显示

## 问题目标

- 当前蓝牙串口使用 `9600` 波特率，发送和接收数据时容易感觉响应不够及时。
- 先提升串口波特率，降低单字节传输时间。

## 改动内容

- 将 `BLUETOOTH_BAUDRATE` 从 `9600U` 修改为 `115200U`。
- 将蓝牙调试页面显示从 `USART2 9600 Ready` 更新为 `USART2 115200 Ready`。

## 注意事项

- 外部蓝牙模块或上位机串口工具也需要设置为 `115200`，否则会出现乱码或无法通信。

## 验证结果

- 已执行 `cmake --build build/Debug`，编译和链接通过。
- 构建结果：RAM 使用 `3232 B / 32 KB`，FLASH 使用 `21864 B / 128 KB`。
