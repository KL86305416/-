# 项目日志：降低 OLED 刷新频率

- 时间：2026-05-13 13:34
- 改动范围：OLED 状态刷新周期配置

## 问题目标

- OLED 通过 I2C 阻塞刷新，刷新过频繁会占用主循环时间。
- 降低 OLED 刷新频率，减少对小车控制响应的影响。

## 改动内容

- 将 `LINE_FOLLOWER_OLED_UPDATE_MS` 从 `100U` 调整为 `300U`。
- 循迹状态页和蓝牙调试页都会使用新的刷新间隔。

## 验证结果

- 已执行 `cmake --build build/Debug`，编译和链接通过。
- 构建结果：RAM 使用 `3232 B / 32 KB`，FLASH 使用 `21864 B / 128 KB`。
